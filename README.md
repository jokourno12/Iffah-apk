# Command
1. install android SDK
2. install flutter
    - checking: flutter doctor
3. flutter create iffah_apk
4. cd iffah_apk
5. flutter run
6. get .apk file: 
    - flutter build apk
    - file in build/app/outputs/flutter-apk
